> **不要删除U盘内的任何文件。**
>
> 有接收器版的键盘只更新接收器固件即可。
>
> 左右手更新固件不需要区分先后顺序。
> 右手键盘更新固件也需要连接USB线。

1. `setting_reset-ncie!nano_v2-zmk.uf2`用于清空键盘内的固件和左右键盘连接的配置信息。
   如果需要左右键盘重新配对，则需要先刷入这个固件。
1. 将键盘通过**USB线**连接到电脑上。
1. 然后**双击RESET键**使键盘进入BootLoader模式。此时键盘会变成一个U盘图标。
1. 只需要把新固件放入这个U盘的**首层目录**即可。
